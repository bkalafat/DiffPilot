# Change Log

All notable changes to the "DiffPilot" extension will be documented in this file.

## [1.0.0] - 2025-12-06

### Added
- Initial release
- **PR Review Tools**
  - `get_pr_diff` - Fetch raw diff between branches
  - `review_pr_changes` - AI-powered code review with diff
  - `generate_pr_title` - Generate conventional PR titles
  - `generate_pr_description` - Generate PR descriptions with checklist
- **Developer Productivity Tools**
  - `generate_commit_message` - Generate commit messages from changes
  - `scan_secrets` - Detect API keys, passwords, tokens
  - `diff_stats` - Get detailed change statistics
  - `suggest_tests` - AI-recommended test cases
  - `generate_changelog` - Generate changelog from commits
- SCM panel integration
- Configuration options for styles and defaults
- Output channel for detailed logs
- Status bar indicator
